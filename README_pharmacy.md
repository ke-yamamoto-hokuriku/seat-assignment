# 北陸大学薬学部 座席指定システム

## 概要
薬学部キャンパスの試験座席指定システム。実測に基づく固定着席マップ方式で、教室ごとに「どの椅子に座れるか」が事前に決定されている。太陽が丘キャンパスのルールベース自動計算方式（`index.html`）とは別システム。

## バージョン履歴
- **Ver.1.0** (2026.04.09) — 初版リリース。202PN, 301P対応

## 太陽が丘システムとの違い

| | 太陽が丘（index.html） | 薬学部（pharmacy_seating.html） |
|---|---|---|
| 着席位置の決定 | ルールベース自動計算 | 実測ベースの固定マップ |
| データ形式 | 物理椅子＋着座ルール | 物理椅子＋着席可能位置リスト |
| 配席方式 | 自動のみ | 自動＋手動（クリック配置） |
| 座席調整 | 移動・交換・詰め直し | 移動・交換 |

## 対応教室

| 教室 | 形状 | ブロック | 列数 | 着席可能数 | 備考 |
|------|------|---------|------|-----------|------|
| 202PN | 扇型 (fan) | 左・中央・右 | 9・11・9 | 46席 | 列ごとに椅子数が変化 |
| 301P | 長方形 (rect) | 左・中央・右 | 11・11・11 | 77席 | 全列同一構成 |

## 教室データ構造（rooms_pharmacy.js）

```javascript
ROOMS[教室ID] = {
  id:     "教室名",
  type:   "fan" | "rect",         // 扇形 or 長方形
  blocks: [                        // ブロック（通路で区切られた塊）
    {
      name:  "ブロック名",         // "左", "中央", "右" 等
      rows: [                      // 列（前方から順）= 1テーブル
        {
          chairs:  数字,           // テーブルの物理的な椅子の総数
          allowed: [番号, ...]     // 着席可能な椅子番号（左から1始まり）
        }
      ]
    }
  ]
}
```

### データの意味
- **1列 = 1テーブル**（ブロック内の横一列の机）
- **chairs** = そのテーブルの物理的な椅子の総数
- **allowed** = 試験時に着席可能な椅子の位置（左から1始まり）。空配列 = その列は着席なし
- **type** = `"fan"`（扇形）で通路が斜め、`"rect"`（長方形）で通路が垂直

### 教室の追加方法
`rooms_pharmacy.js` に同じ形式でデータを追加するだけ。HTMLの編集は不要。教室ボタンは自動生成される。

## 機能

### 名簿読込
- UNIPAの履修者Excelファイル（.xlsx）を読み込み
- 「学籍番号」「学生氏名（名前）」列を自動検出
- 科目名・担当教員名をExcelから抽出して表示
- 学生数と着席可能席数を比較し、不足があれば赤字で警告

### 配席方式（3種類）
- **自動配置（左から）**: 座席通し番号1から順に学生を配置
- **自動配置（右から）**: 座席通し番号を逆順に配置
- **手動配置**: 名簿から学生をクリック → 席をクリックして1人ずつ配置

### 座席通し番号の順序
左ブロック（前→後）→ 中央ブロック（前→後）→ 右ブロック（前→後）

### 座席調整
- 配席済みの席をクリック → 別の席をクリックで**交換**（両方配席済み）または**移動**（空き席へ）
- 変更した席は**赤枠**で表示
- **↩ 取消**: 直前の移動・交換を元に戻す
- **✓ 確定**: 赤枠を消して変更を確定

### Skip設定
- **前 skip**: 前方からN列を使用しない
- **後 skip**: 後方からN列を使用しない
- **左 skip**: 各列の左からN席を使用しない
- **右 skip**: 各列の右からN席を使用しない
- 変更すると配席が自動的に再計算される

### 出力
- **📷 画像DL**: 座席表をPNG画像（2倍解像度）としてダウンロード

### 表示
- オレンジ色 = 着席可能な席（通し番号表示）
- グレー = 着席不可の椅子（小さく表示）
- 白枠＋オレンジ文字 = 配席済み（座席番号＋学籍番号の2段表示）
- 赤枠 = 移動・交換した席（未確定）
- 扇形教室は通路が直線的な扇形に、長方形教室は垂直に表示

## 着席位置の決め方について
薬学部の着席位置は、実際に教室で椅子に座り、隣の答案用紙が見えない位置を確認して決定されたもの。ルールベースの自動計算ではなく、教室ごとに固有の着席マップとして管理している。新しい教室を追加する際は、同様の実測を行い、`rooms_pharmacy.js` にデータを追加する。

## 変更時の注意事項
- **教室の追加**: `rooms_pharmacy.js` にデータ追加のみ。HTMLの編集不要
- **描画コードの変更は慎重に**: セルサイズ・座標計算・SVG生成ロジックの変更はレイアウト崩壊のリスクが高い
