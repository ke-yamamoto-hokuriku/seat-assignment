## 概要
UNIPAの履修者Excelファイルを読み込み、教室レイアウトに学生を配置する座席指定システム。

GitHub Pages: `https://ke-yamamoto-hokuriku.github.io/seat-assignment/`

## 設計原則

### 物理データ起点
- 全教室の物理椅子データを `rooms_physical.js` に定義
- `tables[行][ブロック] = [テーブルサイズ配列]` の3次元構造
- 教室タイプ: fan（扇型）、rect（長方形）、individual（独立机）

### 1席空けルール
- テーブル内の着席数 = `ceil(椅子数 / 2)`
- 着席椅子（座席）と空き椅子（ギャップ）が交互に配置

### ブロック対称配置（b%2ルール）
各ブロックペアが間の通路に向かって寄り合う配置:
- **偶数インデックスブロック（B1, B3, B5）**: 右寄せ、S開始を右端に
- **奇数インデックスブロック（B2, B4, B6）**: 左寄せ、S開始を左端に
- 通路側の椅子が必ず座席になる

### 可変幅列レンダリング
- 座席列: SW=56px（白/黄色矩形、座席番号、学籍番号表示）
- ギャップ列: GW=12px（小さい灰色矩形）
- 通路: AG=20px（行番号表示）
- 座席番号は列優先・上から下の順で自動付与

### 中央4席除外（201・202のみ）
- 中央セクションのB3/B4で椅子数≤2の行にある座席を除外するオプション
- チェックボックスで切り替え、除外席は×マークで表示
- 201: R3/R4の4席、202: R1/R2の4席

## 機能

### 座席配置オプション
- **詰め方**: 横（行優先）/ 縦（列優先）
- **左右**: 左から / 右から
- **並び方**: 順番（名簿順）/ 逆順（名簿逆順）/ 奇数→偶数 / 偶数→奇数
  - 奇数/偶数は**学籍番号の末尾の数字**で判定
- **列間隔・行間隔**: 使用する列/行の間隔
- **前/後/左/右 skip**: 使用しない行・列の数
- **中央4席除外**: 201・202のみ表示されるチェックボックス

### SVG座席表
- 教卓ボックス: 教室名
- 中央セクション上部: UNIPAファイル読込後は科目名・担当者名を表示
- 左右セクション上部: 「正面 左」「正面 右」
- 通路に行番号を表示

### 入出力
- **入力**: UNIPAの履修者Excelファイル（.xlsx）
  - 「学籍番号」「学生氏名」列を自動検出
  - 科目名・担当教員名をヘッダー・座席表に表示
- **出力**: 画像生成（PNG）→ 右クリック/長押しで保存、またはダウンロードボタン

## 技術構成

### 依存ライブラリ（CDN）
- React 18.2.0
- ReactDOM 18.2.0
- SheetJS (xlsx) 0.18.5

### ファイル構成
```
seat-assignment/
├── index.html          ← アプリ本体（UI＋ロジック、JSX事前コンパイル済み）
├── rooms_physical.js   ← 教室物理データ＋グリッド生成関数
└── README.md           ← このファイル
```

### デプロイ
- `index.html` + `rooms_physical.js` の2ファイル構成
- GitHub Pagesに配置するだけで動作（Babel不要）
- ローカルでもブラウザで直接開いて動作
- 学生データはブラウザ内で完結（サーバー送信なし）
