# 北陸大学座席指定システム

## 概要
UNIPAの履修者Excelファイルを読み込み、教室レイアウトに学生を配置する座席指定システム。
GitHub Pages: `https://ke-yamamoto-hokuriku.github.io/seat-assignment/`

## 設計原則

### 教室レイアウト
- 扇型教室（201, 202）は物理形状を`SHAPE`配列で定義し、座席番号は列優先・上から下の順で自動付与
- 長方形教室（302, 404, 406）は`rectGrid(行数, 列数)`で自動生成
- グリッド値: `null`=机なし、`0`=机あり番号なし、`1以上`=番号付き座席

### 通路
- 201と202は5本の通路（セクション境界2本＋セクション内部3本）
- 302, 404, 406は2本（セクション境界のみ）
- 通路位置に行番号を表示

### 座席間の空き
- 番号付き座席の間に1席分の空きスペース（小さい灰色の四角で表現）

## 機能

### 座席配置オプション
- **詰め方**: 横（行優先）/ 縦（列優先）
- **左右**: 左から / 右から
- **並び方**: 順番（名簿順）/ 奇数→偶数 / 偶数→奇数
  - 奇数/偶数は**学籍番号の末尾の数字**で判定
- **列間隔・行間隔**: 使用する列/行の間隔
- **前/後/左/右 skip**: 使用しない行・列の数
- **中央4席除外（201, 202）**: 中央の狭い1人がけ席を除外するチェックボックス
  - 201: 126,127,138,139
  - 202: 73,74,84,85

### 入出力
- **入力**: UNIPAの履修者Excelファイル（.xlsx）
  - 「学籍番号」「学生氏名」列を自動検出
  - 科目名・担当教員名もヘッダーに表示
- **出力**: 画像生成（PNG）→ 右クリック/長押しで保存、またはダウンロードボタン

## 技術構成

### 依存ライブラリ（CDN）
- React 18.2.0
- ReactDOM 18.2.0
- SheetJS (xlsx) 0.18.5

### デプロイ
- `index.html`（アプリ本体）+ `rooms.js`（教室データ）の2ファイル構成（JSX事前コンパイル済み、Babel不要）
- GitHub Pagesに配置するだけで動作
- 学生データはブラウザ内で完結（サーバー送信なし）
